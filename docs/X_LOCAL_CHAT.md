# X Local Chat

The local chat bridge lets Otis talk to X Native tools from one terminal session.

## Start

Run from the repo root:

python scripts\xv7_chat.py

## Commands

Inside the chat loop:

state
diagnose
readiness
proof
package
package apply
apply next
help
exit

## Package workflow

Use package to paste a structured X Prompt Package for preview.

Use package apply to paste and apply a structured package.

End pasted packages with:

ENDX

## Purpose

This is the first bridge away from separate PowerShell commands and toward conversation-driven X operation.

The next bridge can connect the existing web chat or API to the same prompt inbox.
